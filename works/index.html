2017 年度 Pascal 風言語のコンパイラ

大学の演習課題で Pascal のサブセット言語のコンパイラを Java で実装しました。 字句解析器 (Lexer)、構文解析器 (Parser)、意味解析器 (Checker)、コンパイラ (Compiler) を、それぞれゼロから実装していきました。

演習課題という性質のため、ソースコードは非公開としています。

2016 年度 コンウェイのライフゲーム

大学の演習課題で、コンウェイのライフゲームをシミュレートする GUI アプリケーションを実装しました。 Model-View-ViewModel (MVVM) アーキテクチャに基づいた設計になっています。

演習で配られた資料が良くできていて、資料に沿って実装していくだけで MVVM パターンの GUI アプリケーションの実装方法を学べて感動した覚えがあります。

演習課題という性質のため、ソースコードは非公開としています。
